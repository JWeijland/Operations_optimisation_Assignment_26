Implement an existing Mixed Integer Linear Programming model (the model should have some integer of binary elements) by taking the following steps:
  • Select a type of (Mixed) Integer (Linear) Programming problem. Make an appointment if you want something else than:
    • Vehicle routing
    • Gate planning 
    • Runway scheduling or allocation
  • Write down the mathematical formulation and describe the objective and function of the constraints in your own words.
  • Program the model in Excel/VBA/Python or another language. As you will probably need to process quite some data (input and output), a link to Excel or even a database is highly recommended.
  • Verification: Create relatively simple test data sets for which you know what the outcome of the model should be.
  • Create a large scenario, in which you change the data set in such a way that the outcome (should) change along a range of values (sensitivity analysis). Think about how you can present your results.
  • Present the resulting behaviour of the model in a graph.
  • Write down your conclusions about the usability and limitations of the model.

Tips:

  • Use CPlex of Gurobi (Free academic versions available) to optimize the problem.
    • https://community.ibm.com/community/user/datascience/blogs/xavier-nodet1/2020/07/09/cplex-free-for-students
    • https://www.gurobi.com/downloads/end-user-license-agreement-academic/
  • I (Paul) generally use code to a text file with the LP model, but you can also use the callable libraries in both Cplex and Gurobi (Alessandro). Exporting an LP file in that case will help you debug. It will also allow you to run it in a different solver.
  • Please give constraints and variables meaningful names to make processing results and debugging easier.
